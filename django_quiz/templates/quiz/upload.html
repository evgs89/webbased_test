Uploading your test

File format for uploading:
You need to prepare zip-archive with this data (no folders):
test.txt
FIle containing questions and answers, marked with special tags. Each tag must be in beginning of the line.
  • <%%ID> - Number or tag of question
  • <%%Q> - Text of your question
  • <%%QP> - Filename of picture embedded to your question. This file must be in archive too. Optional
  • <%%AC> - Text of correct answer for your question
  • <%%A> - Text of incorrect answer for your question
  • <%%AP> - Filename of picture for current answer. Optional
description.txt
File with description and name of the test. Must contain three lines.
  • url name, latin, must not contain spaces
  • quiz name to show in list
  • description of the quiz
If you need pictures in your quiz place picture files in archive too, without folders
